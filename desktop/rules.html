قوانین و مقررات

در Flutter متن قوانین از API public_content/rules دریافت و به‌صورت متن بلند نمایش داده می‌شود.

Server Content

این صفحه برای نمایش متن رسمی قوانین و مقررات داتملک طراحی شده است. در محصول واقعی، متن این بخش از سرور دریافت می‌شود و باید بدون تغییر محتوایی نمایش داده شود.

چارچوب استفاده از خدمات

کاربر مسئول صحت اطلاعاتی است که در آگهی، درخواست، حساب کاربری، پیام‌ها و فرم‌های پشتیبانی وارد می‌کند. اطلاعات مکانی، فایل‌های تصویری و محتوای آگهی باید با واقعیت ملک و قوانین انتشار داتملک هماهنگ باشند.

حریم خصوصی و موقعیت مکانی

نمایش موقعیت ملک می‌تواند به‌صورت تقریبی انجام شود تا حریم خصوصی مالک و مشاور حفظ شود. ارائه اطلاعات دقیق‌تر باید در مسیرهای امن مثل چت، تماس یا هماهنگی بازدید انجام شود.

گزارش تخلف و پشتیبانی

کاربر می‌تواند محتوای نامعتبر، آگهی مشکوک یا مشکل نرم‌افزاری را از مسیرهای پشتیبانی و تیکت پیگیری کند.
